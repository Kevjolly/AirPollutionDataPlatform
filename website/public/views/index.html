Hello, citizen!

Here, you can get air pollution data from any location where devices have been registered on the data platform. We currently have data from the Bristol city area. Data comes from sources like the Smart Citizen project, Luftdaten or Open Bristol Data. More information can be found in the about section.

You can get data from any location. The platform checks for data from devices around that location. You can also check the data from any device available.
